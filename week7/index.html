Notes

  1. IIFE - An anonymous function that, as the name suggests, is invoked as soon as its defined
  2. Resolved ― the asynchronous operation was completed successfully.
  3. Rejected ― the asynchronous operation didnt work as expected, wasn't successfully completed or resulted in an error.
  4. Ajax is a technique that allows web pages to communicate asynchronously with a server, and it dynamically updates web pages without reloading.
  5. The blob() method is used to read a file of raw data, such as an image or a spreadsheet. Once it has read the whole file, it returns a promise that resolves with a blob object.
